
# AI Agent System (Simple Version)

## How to Run
1. Install dependencies:
   pip install fastapi uvicorn

2. Start the server:
   uvicorn app:app --reload

3. Test endpoint:
   POST /orchestrate
   { "message": "my router not working" }

## Project Includes
- 11 AI Agents
- Knowledge Base
- Orchestrator
- Verification Logic
